# CYSCOM Vulnerable Web Application 

## Setup Instructions

1. use virtual environment:
```b
python -m venv venv
venv/bin/activate 
```

2. Install dependencies:
```
pip install -r requirements.txt
```

3. Run the application:
```
python app.py
```

The application will be available at `http://localhost:5000`
